# XSS

  • POC：全称 ’ Proof of Concept '，中文 ’ 概念验证 ’ ，常指一段漏洞证明的代码。
  • EXP：全称 ’ Exploit '，中文 ’ 利用 '，指利用系统漏洞进行攻击的动作。
  • Shellcode：简单翻译 ’ shell 代码 '，是 Payload 的一种。Shellcode 也有很多种，包括正向的，反向的，甚至 meterpreter。

# 常见 POC

# 反射型 xss

  • 反射型 XSS 的 JS 代码在 web 应用的参数（变量）中，浏览器没有对恶意代码进行过滤
  • 当用户访问带有 XSS 代码的 URL 请求时，服务器端接收请求并处理，然后将带有 XSS 代码的数据返回给浏览器，浏览器解析该段带有 XSS 代码的数据并执行，整个过程就像一次反射，故称为反射型 XSS。
  • 给参数传入 POC 就能验证
