# 1. 命令：

# 基础命令：
main cp 查看 cp 命令的用法
who 列出用户信息
date 显示时间
passwd 修改用户密码
clear 清屏
ls -l 查看当前目录的文件
sudo su 超级管理员
Tab 键 自动补全

# 文件命令
cat 打印文件内容；连接文件
more 显示一屏，按空格显示下一屏
head 显示文件前几行 (默认 10 行)
tail 显示文件后几行
Ctrl + Alt + t 快捷键调出控制台
cp -i file1.c myfile.c 将 file.c 复制到 myfile.c (-i 为覆盖目标目录，有提示)
rm 删除文件

# 目录
cd . . 进入父目录
ls 列出当前目录
mkdir filename 创建目录
chmod mode filename mode 由 3 位 8 进制数字组成
rmdir 删除目录
mkdir 创建目录

# vim

移动光标：左h 右l 上k 下j 在命令方式下

# Linux shell 程序设计

  • 管道符号 “|”
    command 1 | command 2 它的功能是把第一个命令command 1执行的结果作为command 2的输入传给command 2
  • 权限设置：
    chmod a+x ex2
